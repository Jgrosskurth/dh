# Drunken Hitmen - Men's Slowpitch Softball

Est. 2001 | 25+ Years | Legacy Softball League

## Live Site

https://jgrosskurth.github.io/dh/

## How It Works

This is a single-file website (`index.html`) hosted on GitHub Pages. Everything (HTML, CSS, JS, data) is self-contained in one file.

## Updating

- **Stats:** Edit the `battingData` and `pitchingData` arrays in the `<script>` section of `index.html`
- **Schedule:** Edit the `scheduleData` array in the `<script>` section of `index.html`
- **Hall of Fame:** Edit the `hofData` array in the `<script>` section of `index.html`

Push your changes to `main` and GitHub Pages updates automatically.
